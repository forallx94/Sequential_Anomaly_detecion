# Sequential Anomaly detecion

Anomaly detection for Sequential dataset

## Model

* Autoencoder
  * LSTM Autoencoder
  * NN Autoencoder 
* Isolation Forest
* One-Class SVM
* Autoencoder + Isolation Forest
* Unsupervised log message anomaly detection (2 Autoencoder + Isolation Forest)
